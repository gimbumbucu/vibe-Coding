# CLAUDE.md

## 프로젝트 개요
순수 HTML/CSS/JavaScript로 구현하는 웹 프로젝트

---

## 기술 스택

- **HTML5** — 마크업
- **CSS3** — 스타일링 (Bootstrap 5 CDN 사용)
- **JavaScript (ES6+)** — 로직 처리
- 외부 라이브러리 **사용 금지** (Bootstrap 제외)

---

## 네이밍 컨벤션

- 모든 파일명은 **소문자** 사용
- 단어 구분은 **하이픈(-)** 사용
  - ✅ `main-style.css`, `main-app.js`
  - ❌ `mainStyle.css`, `MainApp.js`
- HTML id/class도 동일하게 하이픈 방식 적용
  - ✅ `id="result-area"`, `class="item-card"`
  - ❌ `id="resultArea"`, `class="itemCard"`

## CSS 규칙

- **Bootstrap 5** CDN으로 로드 (로컬 파일 금지)
- 커스텀 스타일은 `css/` 폴더에 작성
- Bootstrap 유틸리티 클래스를 최대한 활용하고, 커스텀 CSS는 최소화
- 색상 등 공통 값은 CSS 변수(`:root`)로 관리

```css
:root {
  --color-primary: #2c3e50;
}
```

- `<link>` 순서: Bootstrap CDN → 커스텀 CSS

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">
<link rel="stylesheet" href="css/style.css">
```

## JavaScript 규칙

- 함수는 **단일 책임 원칙** 적용 — 하나의 함수는 하나의 역할만
- 함수/변수명은 역할이 명확한 **영어 camelCase** 사용
- 매직 넘버는 반드시 **상수(const)**로 선언
- 블록 간 **빈 줄**로 논리 단위 구분
- 복잡한 로직에는 **한글 주석** 작성
- `var` 사용 금지 — `const` / `let` 만 사용
- DOM 조작은 함수로 분리하여 재사용성 확보
- 하나의 함수에 **50줄 이상** 작성 금지 — 분리할 것
- `<script>` 태그는 `</body>` 직전에 배치

# 로또 번호 추천 프로그램 — 기능 명세서

---

## 개요

제외 번호 / 지정 번호 옵션을 설정하고 버튼을 누르면  
추천 번호 5줄을 화면에 출력하는 웹 프로그램

---

## 비즈니스 규칙

- 번호 범위: **1 ~ 45**
- 한 줄당 **6개** 선택, 중복 불가
- 출력: **5줄** 고정
- 각 줄의 번호는 **오름차순** 정렬

---

## 옵션

### 제외 번호
- 생성 결과에 절대 포함되지 않는 번호
- 여러 개 입력 가능

### 지정 번호
- 5줄 모두에 반드시 포함되는 번호
- 최대 **5개** (6개 이상이면 나머지 번호 채우기 불가)
- 제외 번호와 겹치면 에러

### 입력 방식 (공통)
- 숫자 입력 → `Enter` 또는 `+` 버튼으로 태그 추가
- 태그는 개별 삭제 가능

## 디자인

- 전체 배경: `#f8f9fa` (밝은 흰색 계열)
- 카드 배경: `#ffffff`, `box-shadow` 적용
- 번호 공: 원형, `44px × 44px`, bold
- 버튼: Bootstrap `btn-primary`, 둥근 모서리
- 에러 메시지: Bootstrap `alert-danger`

## 유효성 검사

| 상황 | 처리 |
|------|------|
| 1~45 범위 밖 숫자 | 입력 차단 |
| 숫자가 아닌 값 | 입력 차단 |
| 이미 추가된 번호 재입력 | 추가 차단 |
| 지정 번호 6개 이상 | 추가 차단 |
| 지정 번호 ↔ 제외 번호 중복 | 생성 차단 + 에러 |
| 제외 번호가 많아 풀 부족 | 생성 차단 + 에러 |

## 함수 목록

```js
// 번호 생성
pickNums(excludes, fixed)     // 게임 1줄(6개) 생성 → 배열 반환
pickGames(excludes, fixed)    // pickNums 5회 호출 → 2차원 배열 반환

// 스타일
ballClass(num)                // 번호 → CSS 클래스명 반환

// DOM
renderGames(games)            // 결과 영역 렌더링
addTag(num, type)             // 태그 추가   type: 'exclude' | 'fixed'
removeTag(num, type)          // 태그 제거
resetAll()                    // 전체 초기화

// 유효성
validate(num, type)           // 입력값 검증 → { ok, message }
```

---

## 초기화 동작

`resetAll()` 호출 시:
- 제외 번호 / 지정 번호 태그 전체 삭제
- 결과 영역 초기화
- 에러 메시지 초기화

---

## 접근성

- 버튼: `aria-label` 적용
- 번호 공: `aria-label="번호 N"` 적용
